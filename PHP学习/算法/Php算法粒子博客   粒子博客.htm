数据表data：顶级分类为0

cid    cupid  name

1        0         cOne

2        0         cTwo

3        1          cThree

4        2         cFour

5        3         cFive

分类表类似：

…cOne

…|…cThree

…|…|…cFive

…cTwo

接下来：

1.调出数据

$data=array( -1 => array());

while($result=mysql_query(“select * from tablename”)){

$data[$result['fid']]=$result;

}

2.对$data进行处理

foreach($data as $key => $value){

if(array_key_exists($value['fupid'],$data)){

$data[$value['fupid']].['next']=$key;

}

}

3.输出

foreach($data as $key => $value){

if($value['fupid']==0){

echo “|···$value['name']“;

$num=1;

$next=$value['next'];

while($next){

for($i=0;$i<$num;$i++) echo “|···”;

}

}

组合算法的编程实现

2011 年 06 月 21 日 16:09 ⋅ Leave a Comment ⋅ helion
Number of View: 57

思路：

本程序的思路是开一个数组，其下标表示1到m个数，数组元素的值为1表示其下标
代表的数被选中，为0则没选中。
首先初始化，将数组前n个元素置1，表示第一个组合为前n个数。
然后从左到右扫描数组元素值的“10”组合，找到第一个“10”组合后将其变为
“01”组合，同时将其左边的所有“1”全部移动到数组的最左端。
当第一个“1”移动到数组的m-n的位置，即n个“1”全部移动到最右端时，就得
到了最后一个组合。
例如求5中选3的组合：
1   1   1   0   0   //1,2,3
1   1   0   1   0   //1,2,4
1   0   1   1   0   //1,3,4
0   1   1   1   0   //2,3,4
1   1   0   0   1   //1,2,5
1   0   1   0   1   //1,3,5
0   1   1   0   1   //2,3,5
1   0   0   1   1   //1,4,5
0   1   0   1   1   //2,4,5
0   0   1   1   1   //3,4,5

算法研究：求$b相对于$a的相对路径

2011 年 06 月 21 日 16:00 ⋅ Leave a Comment ⋅ helion
Number of View: 58

新浪面试过的朋友提供的这个题目：

$a = ‘/a/b/c/d/e.php’; $b = ‘/a/b/12/34/c.php’; //计算出 $b 相对于 $a 的相对路径应该是 ../../c/d

php面试题的答案：
function getRelative($a,$b) {
$arr = explode(“/”,$a);
$brr = explode(“/”,$b);
$c = count($arr)-2;
$d = count($brr)-2;
//之所以减二，一个是不在后面的文件名，
//一个是数组的索引是从0开始的，比数组中的第一维的个数要小一
$e = ($c>$d) ? $c:$d;
$str1 = $str2 = ”;
for ($j=0;$j<=$e;$j++) {
$cur_a = isset($arr[$j]) ? $arr[$j] : ”;
$cur_b = isset($brr[$j]) ? $brr[$j] : ”;
if ($cur_a == $cur_b) {
continue;
} else {
if ($j <= $c)
{
$str1.=’/’.$cur_a;
}
if ($j <= $d )
{
$str2.=”../”;
}
}
}
return $str2.substr($str1,1,strlen($str1));
}
