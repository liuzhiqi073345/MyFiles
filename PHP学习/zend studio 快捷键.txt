Ctrl + Shift + /　块注释 
Ctrl + U　        选择的代码片段全部转换为大写 
Ctrl + L　        选择的代码片段全部转换为小写 
Ctrl + D　        复制光标所在行 
Ctrl + E　        删除光标所在行 
Tab　             增加代码缩进 
Shift + Tab　     减少缩进 
Ctrl + Alt+ F　   在文件中查找 
Ctrl + BackSpace　删除光标前一个单词或一个符号 
Ctrl + G　        转到行 
Alt + →　        定位到光标的下一个位置 
Alt + ←　        定位到光标的上一个位置

环境切换 
Esc　            隐藏辅助窗口 
Ctrl + W　       切换自动换行 
Ctrl + Tab　     在各个编辑器标签页之间切换 
Ctrl + F4　      关闭当前标签页 
Alt + F4　       关闭ZDE

Ctrl + ~　       空格

Ctrl + Shift + F2　删除所有书签 
Ctrl + Shift + M　 显示书签对话框

网上搜索的:

Ctrl+1 快速修复(最经典的快捷键就不用多说了)

Ctrl+D: 删除以后行

Ctrl+Alt+↓ 复制以后行到下一行(复制增加)

Ctrl+Alt+↑ 复制以后行到上一行(复制增加)

Alt+↓ 以后行和下面一行交互位置(特别实用可以省去先剪切再粘贴了)

Alt+↑ 以后行和上面一行交互位置(同上)

Alt+← 前一个编辑的页面

Alt+→ 下一个编辑的页面(当然是针对上面那条来说了)

Alt+Enter 显示以后选择资源(工程or 文件 or文件)的属性

Shift+Enter 在以后行的下一行插入空行(这时鼠标可以在以后行的任一位置不一定是最后)

Shift+Ctrl+Enter 在以后行插入空行(原理同上条)

Ctrl+Q 定位到最后编辑的地方

Ctrl+L 定位在某行 (对于程序超过100的人就有福音了)

Ctrl+M 最大化以后的Edit或View (再按则反之)

Ctrl+/ 注释以后行再按则取消注释

Ctrl+O 快速显示 OutLine

Ctrl+T 快速显示以后类的继承结构

Ctrl+W 关闭以后Editer

Ctrl+K 参照选中的Word快速定位到下一个

Ctrl+E 快速显示以后Editer的下拉列表(如果以后页面没有显示的用黑体表示)

Ctrl+/(小键盘) 折叠以后类中的所有代码

Ctrl+×(小键盘) 展开以后类中的所有代码

Ctrl+Space 代码助手完成一些代码的插入(但一般和输入法有冲突可以修改输入法的热键也可以暂用Alt+/来代替)

Ctrl+Shift+E 显示管理以后打开的所有的View的管理器(可以选择关闭激活等糙作)

Ctrl+J 正向跃羁查找(按下Ctrl+J后你所输入的每个字母编辑器都提供快速匹配定位到某个单词如果没有则在stutes line中显示没有找到了查一个单词时特别实用这个功能Idea两年前就有了)

Ctrl+Shift+J 反向跃羁查找(和上条相同只不过是从后往前查)

Ctrl+Shift+F4 关闭所有打开的Editer

Ctrl+Shift+X 把以后选中的文本全部变为大写

Ctrl+Shift+Y 把以后选中的文本全部变为小写

Ctrl+Shift+F 格式化以后代码

Ctrl+Shift+P 定位到对于的匹配符(譬如{}) (从前面定位后面时光标要在匹配符里面后面到前面则反之)
